Crypto Playground

Experiment with classic ciphers: Caesar, ROT13, XOR, and Base64.

(^.^) These are educational ciphers. Do not use for real secrets. Use modern crypto libraries for real security.

Output

Examples
  • Caesar shift 3: "HELLO" → "KHOOR"
  • ROT13: "Hello" → "Uryyb" (same for encrypt/decrypt)
  • XOR with key "42": Encoded text in Base64
  • Base64: "Sachin" → "U2FjaGlu"
